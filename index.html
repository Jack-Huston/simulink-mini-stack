Overview

If you are hiring for GN&C or modeling and simulation, this project shows how I build a closed-loop Simulink model and test it with noise and disturbances.

Plant and actuation

  • Disturbance injection for robustness

Sensors and noise

  • IMU measurement path with noise terms
  • Star sensor updates with rate effects
  • Bias cases for test runs

Control loop

  • Scenario parameters for repeatable runs
